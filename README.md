# CP3_APP

A static web app presentation for the **SpeakUp** school feedback campaign.

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy to GitHub Pages

1. Push this branch/repository to GitHub.
2. In your repository settings, enable **Pages** and choose **Deploy from a branch**.
3. Select the branch you want to publish (for example `main`) and set folder to `/ (root)`.
4. Save and wait for GitHub Pages to publish.

Your site URL will look like:

`https://<your-username>.github.io/CP3_APP/`
